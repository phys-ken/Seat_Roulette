# 座席ルーレット / Seat Roulette

教室での座席配置をランダム化・グループ分けするWebアプリ集です。  
黒板への投影を想定したシンプルなUIで、ブラウザだけで動作します。インストール不要。

---

| ファイル | 用途 | 席数 |
|---|---|---|
| [layout_7by6.html](https://phys-ken.github.io/Seat_Roulette/layout_7by6.html) | 座席シャッフル（7列×6行） | 最大42席 |

## 1. 座席シャッフル（7×6 / 7×7 / カスタム）

教室の座席をランダムに入れ替えるツールです。黒板に投影しながら使えます。

### 使い方

1. **座席レイアウトの決定**  
   無効にしたい座席をクリック → 赤く暗転して除外されます。
2. **シャッフル開始**  
   「シャッフル開始」ボタンで番号が高速でランダムに回転します。
3. **ストップ**  
   「ストップ」ボタンで確定。有効な座席数ちょうどの生徒番号で配置されます。
4. **全座席有効化**  
   クリックして無効にした座席を一括リセットできます。
5. **Excel ダウンロード**  
   配置結果を `.xlsx` で保存。**生徒目線・教師目線の両方**が1シートに出力されます。

### カスタムレイアウトのみの操作

- 画面右上の **行 / 列** の `+` `-` ボタンで、グリッドサイズをリアルタイムに変更できます。
- 行：1〜12、列：1〜15 の範囲で自由に設定可能。

### Excel出力の仕様

- ファイル名：`zaseki_YYYYMMDDhhmm.xlsx`
- シート構成：上半分が生徒目線、下半分が教師目線（180度回転）
- 教卓の位置が明記されるため、印刷して配布するだけで使えます。

## 2. グループ分けルーレット

出席番号からランダムにグループ分けを行うツールです。結果はPDFで出力できます。

### 使い方

#### 設定画面
1. **出席番号上限** を入力します（例：42人クラスなら42）。
2. **欠席者** をクリックして選択します（背景色が変わります）。
3. **グループ数** と **各グループの人数** を設定します。  
   - 合計人数が在籍人数と一致しないと「ルーレット開始」ボタンは押せません。

#### ルーレット画面
4. 「スタート」ボタンで番号が回転します。
5. 「ストップ」でグループ確定。欠席者は「欠」と表示されます。

#### 結果の確認
結果は3つのタブで確認できます。

| タブ | 内容 |
|---|---|
| 生徒ビュー | 出席番号順にグループ名を表示した一覧表 |
| 班ごとビュー | グループ別のメンバー一覧（列ごとに分割表示） |
| 円卓ビュー | グループを円卓（多角形）で視覚的に表示 |

6. **「もう一度」** で同じ設定のままシャッフルをやり直せます。
7. **「◀ 設定」** で設定画面に戻り、人数やグループを変更できます。
8. **「PDF ⬇」** で3ページ構成のPDFを出力します。

### PDF出力の仕様

- **1ページ目**：生徒ビュー（出席番号 → グループ名の一覧）
- **2ページ目**：班ごとビュー（グループ別メンバー表）
- **3ページ目**：円卓ビュー（グループを円形配置で視覚表示）
- 全ページにタイムスタンプが記載されます。

## 技術仕様

|---|---|
| 実装 | Vanilla JavaScript（フレームワーク不使用） |
| Excel出力 | [SheetJS (xlsx)](https://sheetjs.com/) |
| PDF出力 | [jsPDF](https://github.com/parallax/jsPDF) + [html2canvas](https://html2canvas.hertzen.com/) |
| 動作環境 | モダンブラウザ全般（インストール不要） |
| サーバー | 不要（静的HTMLファイルとして動作） |
