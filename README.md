# 座席ルーレット - Seat Roulette

教室での座席配置を簡単にランダム化できるWebアプリケーションです。

## 🎯 主な機能

- **座席の有効・無効切替**: クリックで任意の座席を無効化
- **スマートシャッフル**: 有効な座席のみでランダム配置
- **順番表示**: シャッフル前に出席番号順で確認可能
- **Excel出力**: 生徒目線・教師目線の両方の座席表をダウンロード
- **黒板投影対応**: プロジェクター表示に最適化された横長レイアウト

## 🚀 アプリを使用する

教室の形状に合わせて自由に設定できる座席ルーレットです。

## 💡 使い方

1. **座席の無効化**: 欠席者や使用しない座席をクリックして無効化
2. **確認**: 有効な座席に出席番号順で配置されることを確認
3. **シャッフル**: 「シャッフル開始」→「ストップ」でランダム配置
4. **出力**: 「確定 & Excel ダウンロード」で座席表を保存

## 📋 Excel出力の特徴

- **生徒目線座席表**: A2:G7範囲に配置
- **教師目線座席表**: A11:G16範囲に180度回転して配置
- **タイムスタンプ**: ファイル名に日時を自動付与
- **教卓表示**: D1(生徒目線)、D17(教師目線)に明記

## 🛠️ 技術仕様

- **フロントエンド**: Vanilla JavaScript
- **Excel出力**: SheetJS (xlsx)
- **レスポンシブ**: プロジェクター投影に最適化
- **ブラウザ対応**: モダンブラウザ全般
