# 座席ルーレット - Seat Roulette

教室での座席配置を簡単にランダム化できるWebアプリケーションです。

## 🎯 主な機能

- **座席の有効・無効切替**: クリックで任意の座席を無効化
- **スマートシャッフル**: 有効な座席のみでランダム配置
- **順番表示**: シャッフル前に出席番号順で確認可能
- **Excel出力**: 生徒目線・教師目線の両方の座席表をダウンロード
- **黒板投影対応**: プロジェクター表示に最適化された横長レイアウト

## 🚀 アプリを使用する

### 📊 7×6レイアウト（42席対応）
標準的な教室レイアウトに対応した座席ルーレットです。

**[📱 7×6座席ルーレットを開く](https://phys-ken.github.io/Seat_Roulette/layout_7by6.html)**

### ⚙️ カスタムレイアウト
教室の形状に合わせて自由に設定できる座席ルーレットです。  
**行数**: 1〜12行、**列数**: 1〜15列まで自由に設定可能！

**[🔧 カスタム座席ルーレットを開く](https://phys-ken.github.io/Seat_Roulette/layout_custom.html)**

#### 🎛️ カスタムレイアウトの特徴
- **可変グリッド**: 1×1から12×15まで自由にサイズ設定
- **リアルタイム調整**: +/-ボタンまたは直接入力で行列数を変更
- **即座に適用**: 「適用」ボタンで瞬時にレイアウト更新
- **最大180席対応**: 12×15の最大構成で180席まで対応
- **動的Excel出力**: 設定したサイズに応じて生徒目線・教師目線を自動配置

## 💡 使い方

### 📋 基本操作
1. **座席の無効化**: 欠席者や使用しない座席をクリックして無効化
2. **確認**: 有効な座席に出席番号順で配置されることを確認
3. **シャッフル**: 「シャッフル開始」→「ストップ」でランダム配置
4. **出力**: 「Excel ダウンロード」で座席表を保存

### 🔧 カスタムレイアウト操作
1. **サイズ設定**: 右上の+/-ボタンで行列数を調整
2. **直接入力**: 数値入力フィールドで正確な値を設定
3. **適用**: 「適用」ボタンでレイアウトを更新
4. **座席管理**: 通常と同様にクリックで座席の有効/無効を切替

## 📋 Excel出力の特徴

### 📊 7×6レイアウト
- **生徒目線座席表**: A2:G7範囲に配置
- **教師目線座席表**: A11:G16範囲に180度回転して配置
- **教卓表示**: D1(生徒目線)、D17(教師目線)に明記

### 🔧 カスタムレイアウト
- **動的サイズ対応**: 設定した行列数に応じて自動配置
- **生徒目線**: 1行目から設定サイズで配置
- **教師目線**: 生徒目線の下部に180度回転して配置
- **教卓表示**: 各レイアウトの中央に配置
- **ファイル名**: `zaseki_custom_{行}x{列}_{タイムスタンプ}.xlsx`

### 📝 共通仕様
- **タイムスタンプ**: ファイル名に日時を自動付与
- **開発者リンク**: Excel内にWebアプリへのハイパーリンク

## 🛠️ 技術仕様

- **フロントエンド**: Vanilla JavaScript
- **Excel出力**: SheetJS (xlsx)
- **レスポンシブ**: プロジェクター投影に最適化
- **ブラウザ対応**: モダンブラウザ全般
